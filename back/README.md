# gestion_association_full

## **BACKEND**

Ce partie est réservée à la partie Back de l'application et du site

Langage: JAVA

#### Framework : Spring Boot.
#### API : Spring Web MVC (REST) ou Spring GraphQL.
#### Sécurité : Spring Security + JWT/Keycloak.
#### Base de données : Spring Data JPA (Hibernate).
#### Cache : Spring Cache + Redis.
#### Temps réel : Spring WebSocket ou Kafka.
#### Tests : JUnit 5 + Mockito + Testcontainers.
#### Documentation : SpringDoc OpenAPI.
#### Déploiement : Docker + Kubernetes.
#### Monitoring : Spring Boot Actuator + Prometheus + Grafana.
