# gestion_association_full

## **APP**

Ce partie est réservée à la partie front de l'application

####  Framework : Next.js ou Remix (optionnel mais recommandé).
#### Routes : Next.js Router
#### Service Worker : Workbox.
#### Manifest : @pwa-manifest ou react-pwa-install.
#### Stockage : idb ou localforage.
#### Synchronisation : Background Sync API ou workbox-background-sync.
#### UI : MUI, Chakra UI, ou Ant Design.
#### Formulaires : React Hook Form + Zod.
#### Build : Vite ou Webpack.
